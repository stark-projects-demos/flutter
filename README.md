# Flutter Demo App

This is a simple demo Flutter application that recreates the TeamSync landing page for accessibility scanning demonstrations.

To test sending results to your [Stark dashboard](https://app.getstark.co), follow the instructions at [https://www.getstark.co/support/getting-started/using-source-code-integrations-for-mobile-frameworks/](https://www.getstark.co/support/getting-started/using-source-code-integrations-for-mobile-frameworks/)

## Getting Started

```bash
# Get dependencies
flutter pub get

# Run on web
flutter run -d chrome
```
